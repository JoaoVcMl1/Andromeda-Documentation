Programação

Nessa seção contém todos os métodos do framework da Andromeda.

JavaScript

JavaScript é a principal línguagem usada na engine, sendo a mais potente e recomendada. Abaixo contém seus métodos e exemplos de como usá-los:

Definir uma imagem

Para definir uma imagem a um sprite, usamos o metodo setImage:

function create() {
    setImage(sprite, textura);
}

Definir uma largura e altura

Para definir uma largura e uma altura a um sprite, usamos o metodo setWidth e setHeight:

function create() {
    setWidth(sprite, largura);
    setHeight(sprite, altura);
}

Definir a camera para um sprite

Para definir a camera a um sprite usamos o método setCamera para apenas definir a camera para o objeto e startFollow para centralizar a camera ao objeto:

function create() {
    setCamera(sprite, scene); //o metodo scene contem a cena do jogo
    startFollow(sprite, scene);
}

Verificar clique

Para verificar se um sprite ou UISprite foi clicado, usamos o método onClick:

function create() {
    onClick(sprite, function() {
   //Evento de clique
   });  
}

Verificar pressão

Para verificar se um sprite ou UISprite foi pressionado, usamos o método onPressed:

function create() {
    onPressed(sprite, 
         function() {
          //quando esta sendo pressionado
},
         function() {
          //quando a pressão acaba
});
}

Definir Velocidade

Para definir uma velocidade horizontal ou vertical a uma sprite usamos os métodos setVelocityX e setVelocityY:

function create() {
    sprite.setVelocityX(5); //define a velocidade horizontal para 5
    sprite.setVelocityY(5); //define a velocidade vertical para 5
}

Definir Rotação

Para definir uma rotação a uma sprite usamos o método setRotation:

function create() {
    setRotation(sprite, 90); //define uma rotação de 90° para o sprite
}

Definir Posição

Para definir uma posição a uma sprite usamos os métodos srpite.x e sprite.y:

function create() {
    sprite.x = 10; //define uma posicao horizontal de 10 pixels para o sprite
    sprite.y = 10; //define uma posicao vertical de 10 pixels para o sprite
}

Criar diretorio de animações

Um diretorio de animações pode ser criado passando o nome das imagens para um array dir = ['imagens'];

function create() {
    const diretorio = [
    'imagem',
    ];
}

Criar animações

Depois do diretorio ser criado e configurado corretamente com as imagens, uma animação pode ser criada usando o método createAnimation:

function create() {
    const diretorio = [
    'imagem',
    ];
    frameRate = 10; //tempo entre animações
    const anim = createAnimation(sprite, diretorio, frameRate);
    //sprite recebe o nome do objeto que a animação sera aplicada
    //diretorio recebe o nome do diretorio criado e configurado com as imagens
}

Iniciar e parar animações

Depois da animação ser criada e configurada corretamente podemos iniciar ou parar uma animação usando os métodos anim.start e anim.stop

function create() {
    const diretorio = [
    'imagem',
    ];
    frameRate = 10;
    const anim = createAnimation(sprite, diretorio, frameRate);
    anim.start(); //inicia a animação
    anim.stop(); //para a animação
}

Destruir objetos

Para destruir um objeto e seu respectivo colisor, é utilizado o método sprite.destroy

function create() {
   sprite.destroy();
}
