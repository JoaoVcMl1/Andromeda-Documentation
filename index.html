Documentação

Bem vindo a documentação da Andromeda Engine, uma Game Engine poderosa para criar jogos 2d e indie, contendo 2 linguagens de programação como base, sendo elas Python e JavaScript.

Categorias

Essa documentação é dívida em categorias, sendo elas:

  • Instalação: Nessa seção contem um tutorial completo de como instalar e começar seu primeiro projeto na engine.
  • Programação: Nessa seção você aprende a usar as 2 linguagens de programação da Game Engine, contendo todos os métodos disponíveis.
